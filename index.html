Dicionário de HTML

Tag de DIV

A tag <div> define uma divisão ou uma seção em um documento HTML. A tag <div> é usada como um contêiner para elementos HTML, que são então estilizados com CSS ou manipulados com JavaScript. A tag <div> pode ser facilmente estilizada usando o atributo class ou id. Qualquer tipo de conteúdo pode ser colocado dentro da tag <div>! Observação: por padrão, os navegadores sempre colocam uma quebra de linha antes e depois do elemento <div>.

Tag de H1 .

As tags <h1> até <h6> são usadas para definir títulos HTML. A tag <h1> define o título mais importante. A tag <h6> define o título menos importante.

Tag de TABLE

A tag <table> é usada para definir uma tabela HTML. Dentro da tabela, usamos as tags <tr> para definir linhas, <th> para definir células de cabeçalho e <td> para definir células de dados.

Tag de IMG

A tag <img> é usada para incorporar imagens em um documento HTML. O atributo src especifica o caminho da imagem, e o atributo alt fornece uma descrição alternativa para leitores de tela.

Tag de A

A tag <a> define um hyperlink que conecta uma página a outra. O atributo href especifica o destino do link.

Tag de FORM

A tag <form> é usada para criar um formulário HTML que pode ser enviado para um servidor. Dentro do formulário, você pode usar tags como <input>, <textarea> e <button>.



Tag de UL e OL

A tag <ul> define uma lista não ordenada, enquanto a tag <ol> define uma lista ordenada. Ambas as tags utilizam a tag <li> para definir itens da lista.

Lista Não Ordenada

  • Item 2
  • Item 3

Lista Ordenada

  1. Item A
  2. Item B
  3. Item C

Tag de SPAN

A tag <span> é usada para aplicar estilos a uma parte específica de texto ou conteúdo dentro de um elemento. Ao contrário da tag <div>, que cria uma divisão de bloco, a tag <span> é um elemento em linha.

Texto normal e texto em vermelho usando a tag <span>.

Tag de BUTTON

A tag <button> define um botão clicável. Pode ser usado em formulários ou para executar scripts.
